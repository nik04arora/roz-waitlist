# Hero header images (for Roz rebrand)

These are the source assets from the live homepage hero, exported as **PNG** for AI image editors (Photoshop, Figma, Midjourney, etc.).

## Which file to edit

| File | What it is | Alinea branding to change |
|------|------------|---------------------------|
| **`hero-phone.png`** | Phone mockup (main focal point) | “Allie” AI chat, purple app UI, Alinea-style home icon |
| **`hero-ui-layer-full.png`** | Wide floating UI cards behind the phone | Pink **A** logo, “daily briefing”, stock-focused copy |
| **`hero-background.png`** | Purple glow / particles (full-width backdrop) | Mostly abstract — optional color tweak for Roz |
| `hero-ui-layer.png` | Same UI layer at 1600px width (smaller) | Use `hero-ui-layer-full.png` instead |

**Start with `hero-phone.png`** if you only edit one image — that’s what visitors notice first.

## After editing

1. Save your new PNG(s) in this folder (e.g. `hero-phone-roz.png`).
2. Tell your agent to swap them into `index.html` (or convert to `.avif` / `.webp` for production).

## Original formats

`.avif` files are the CDN originals (smaller for the web). PNGs are for editing only.

## Bundle

`hero-assets-for-editing.zip` — all PNGs + this README.
